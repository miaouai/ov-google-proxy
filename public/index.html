🚀 三步导入配置

  1. 选择配置：根据需求选择卫星图、街道图或混合图的 XML 配置
  2. 复制配置：全选对应 XML 代码框中的内容（Ctrl+A），然后复制（Ctrl+C）
  3. 导入奥维：
    • 打开奥维地图 → 系统 → 导入对象
    • 选择 "从文本导入"
    • 粘贴刚才复制的 XML 配置
    • 点击 "确定"完成导入
    • 在左侧图层列表中找到并启用对应的 Google 图层
💡 温馨提示：
  • 如果地图加载缓慢，可以尝试切换到其他类型的地图
  • Cloudflare Worker 每月免费额度为 10 万次请求，超出后可能限速
  • 三个配置可同时导入，根据需要切换使用
